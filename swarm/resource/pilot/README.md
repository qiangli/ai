# Pilot (Ad hoc Release) - Agents/Tools/Models

Overview
- Automatically created, temporary configuration snapshot for quick testing of agents, tools, and model prompts/configs. Openly editable by anyone. Intended for single-use experiments.

Scope
- Use for rapid experiments with new agents, tool integrations, or model prompts. Not production-ready.

Lifecycle & Promotion
- Auto-created by the release workflow; writable by anyone; designed for one-off use.
- Reuse across multiple releases or a short-lived persistence signals promotion to Incubator.

Promotion to Incubator
- Criteria (examples):
  - Used in two or more distinct releases within 30 days
  - Involves inputs from multiple teams
  - Active beyond 14 days without a clear single-use conclusion
- When criteria are met, move to the Incubator state.

Notes
- This pilot configuration focuses on agents/tools/models contexts, including agent configs, tool integrations, and model prompts.
