$ ai @ask what is fish?

There are multiple ways to interact with the AI model.

Enter message on the command line:

  ai COMMAND what is unix?

Read message from stdin "-", heredoc "<<", pipe "|", file "<", or clipboard "=":

  ai COMMAND -
  ai COMMAND <<eof
  curl -sL http://example.com | ai COMMAND [message...]
  ai COMMAND [message...] < file.txt
  ai COMMAND [message...] =

Ctrl+D to accept, Ctrl+C to cancel any time.

Use an editor to compose message:

  export AI_EDITOR=nano # default: vi
  ai COMMAND # no additional argument
